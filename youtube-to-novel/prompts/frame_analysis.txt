# 关键帧分析 Prompt 模板
# 使用方式：将 {subtitle} 替换为实际字幕内容后发送给多模态模型

---

## frame_analysis_prompt

```text
你是一位专业的视觉叙事分析师，擅长从影像中提取文学创作素材。

请分析这张视频截图，同时参考以下同时段的字幕/对话内容：

【字幕内容】
{subtitle}

请以 JSON 格式返回分析结果，严格按照以下结构：

{
  "scene": "场景环境描述（地点类型、空间感、光线、色调、时间感）",
  "characters": [
    {
      "description": "人物外貌描述（年龄感、服装、表情、姿态）",
      "emotion": "当前情绪状态"
    }
  ],
  "action": "正在发生的核心动作或事件",
  "atmosphere": "整体氛围和情绪基调（一个词或短语）",
  "narrative_hint": "这个场景暗示的故事走向或深层含义",
  "sensory_details": "可用于小说描写的感官细节（视觉/听觉/嗅觉等）"
}

注意：
- 如果截图模糊或无法识别，scene 填"画面不清晰"，其他字段基于字幕内容推断
- characters 为空数组如果没有人物出现
- 保持客观描述，不要臆断情节
```

## global_summary_prompt

```text
你是一位资深编辑，擅长从素材中提炼故事核心。

以下是一个视频按时间顺序排列的内容分析片段：

{timeline_json}

视频元信息：
- 标题：{video_title}
- 时长：{duration}
- 频道：{channel}
- 描述：{description}

请整合以上内容，生成一份供小说创作使用的素材摘要，以 JSON 格式返回：

{
  "core_story": "核心故事线（100 字以内）",
  "main_characters": [
    {
      "name_or_label": "人物标签（如：男主角/旁白者/受访者）",
      "traits": "性格特点和外貌特征",
      "role": "在故事中的功能"
    }
  ],
  "key_events": [
    "事件 1（按时间顺序）",
    "事件 2",
    "..."
  ],
  "emotional_arc": "情感/氛围的变化曲线（如：平静→紧张→释然）",
  "themes": ["主题 1", "主题 2"],
  "setting": "主要场景和时代背景",
  "suggested_genres": ["最适合的小说类型 1", "类型 2"],
  "writing_opportunities": "最值得在小说中重点展开的场景或细节"
}
```
